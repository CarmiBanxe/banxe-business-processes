# banxe-business-processes — Claude Code Instructions

## Purpose

This repository contains ArchiMate 3.2 business process models for the Banxe EMI platform.
It feeds AI agents with machine-readable process definitions.

## Key directories

- `processes/` — business process XML/JSON definitions
- `exchange/` — Open Exchange XML exports from Archi Tool
- `ai-agent-context/` — pre-processed JSON context for AI agents
- `invariants/` — business invariants (I-01..I-29) as structured data

## Conventions

- All business processes follow ArchiMate 3.2 Open Group standard
- Process IDs: `BP-<domain>-<NNN>` e.g. `BP-PAY-001`
- Actor IDs: `ACT-<NNN>` e.g. `ACT-001`
- Invariant IDs: `I-<NN>` e.g. `I-05`

## Do not edit in this repo

- `diagrams/svg/` and `diagrams/png/` — generated from Archi Tool exports
- `ai-agent-context/*.json` — generated by import scripts
